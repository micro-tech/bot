# HyEvo Evolutionary Engine

HyEvo is the adaptive workflow evolution system in Agent OS.

## Capabilities
- Workflow definition using nodes (Skill, LLM, Code)
- Reflection-based improvement using LLMs
- Population management and selection

- `src/hy_evo/`
- `src/hy_evo/workflow.rs`
- `src/hy_evo/population_manager.rs`
